## Parts
- Arduino Uno
- LCD 16x2 with I2C
- Wires

## Step
1. Upload I2C_Scanner.ino to get LCD address
2. Use LCD_Display_Name.ino to show name

## Wire
- VCC to 5V
- GND to GND
- SDA to A4
- SCL to A5
